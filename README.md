# ClickCheck Project

Проект на Django с использованием Django Channels и Redis для поддержки WebSocket и чата в реальном времени.

## Требования

- Python 3.9 или выше
- Redis сервер (для поддержки WebSocket)
- Виртуальное окружение для Python (рекомендуется)
